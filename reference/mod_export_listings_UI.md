# Create user interface for the export listings shiny module of dv.listings

Create user interface for the export listings shiny module of
dv.listings

## Usage

``` r
mod_export_listings_UI(module_id)
```

## Arguments

- module_id:

  `[character(1)]` A unique ID string to create a namespace. Must match
  the ID of `mod_exp_listings_server()`.

## Value

A shiny `uiOutput` element.
