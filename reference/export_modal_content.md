# Internal helper function that specifies the content of the download modal dialogue.

Internal helper function that specifies the content of the download
modal dialogue.

## Usage

``` r
export_modal_content(ns, file_name, cond, colnames, activate_checkbox = FALSE)
```

## Arguments

- ns:

  A function that expects an ID and returns the namespaced ID, usually
  generated by [`shiny::NS()`](https://rdrr.io/pkg/shiny/man/NS.html).

- file_name:

  `[character(1)]` A string containing the initial file name.

- cond:

  `[character(1)]` A string containing a JavaScript expression that will
  be evaluated to determine whether the conditionalPanel should be
  displayed.

- colnames:

  `[character(0+)]` A character vector containing the reference column
  choices.

- activate_checkbox:

  `[logical(1)]` A boolean indicating whether the checkbox should be
  activated or not. Default is FALSE.

## Value

A shiny tagList.
